About

Asu Software is a company that manufactures mobile games for android. This company was set up in January 2019 and currently has 3 games on the Google Play Store. So Asu Software team wishes you a good welcome to this page.

Download Resume
Xrace
We've created this game to try and challenge each player and set new records. This game is based on speed and reflexes, you have to make sure the walls do not come to you and you get to the end.
Run Boy!
Run Boy is a game where you have to help the little boy to cross the decks without falling down, and so get to the end. This game is based on speed and intelligence.
Parking Sport Cars
This game is a game where you have to park expensive cars and sports without scratching in any kind of cases that you can meet in the real world. This game was designed especially for those who want to learn to park a car and once again have fun playing this game.
